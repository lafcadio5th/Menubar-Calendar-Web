Menubar Calendar

隱私權政策

最後更新：2026 年 1 月

Menubar Calendar（以下簡稱「本應用程式」）非常重視您的隱私。本隱私權政策說明我們如何收集、使用和保護您的個人資訊。

資料收集

本應用程式設計以隱私為優先。我們收集的資料極為有限：

  • 日曆資料：本應用程式讀取您 Mac 上的 Apple Calendar 資料以顯示行程。這些資料僅在本地處理，不會傳送至任何伺服器。
  • 提醒事項：本應用程式讀取 Apple Reminders 資料以顯示待辦事項。這些資料僅在本地處理。
  • 位置資訊（選擇性）：如果您啟用天氣功能，我們會使用您的大致位置（城市層級）來獲取天氣資訊。我們不會儲存您的精確位置。

第三方服務

本應用程式可能使用以下第三方服務：

  • 天氣服務：我們使用第三方天氣 API 提供天氣資訊。僅傳送城市名稱，不傳送個人身份資訊。
  • Todoist（選擇性）：如果您選擇連結 Todoist 帳號，我們會透過官方 API 存取您的任務資料。我們不會儲存您的 Todoist 憑證。

資料儲存

所有設定和快取資料都儲存在您的 Mac 本地。我們不營運任何伺服器來儲存用戶資料。

資料分享

我們不會出售、交易或以其他方式轉讓您的個人資訊給第三方。

安全性

本應用程式使用 macOS 的安全機制來保護您的資料，包括 App Sandbox 和 Keychain 服務。

您的權利

您可以隨時：

  • 在系統偏好設定中撤銷本應用程式的日曆、提醒事項或位置權限
  • 刪除本應用程式以移除所有本地儲存的設定
  • 聯絡我們詢問關於您資料的任何問題

兒童隱私

本應用程式不針對 13 歲以下的兒童，我們不會故意收集兒童的個人資訊。

政策變更

我們可能會不時更新本隱私權政策。任何變更都會在此頁面公布，並更新「最後更新」日期。

聯絡我們

如果您對本隱私權政策有任何疑問，請透過 GitHub Issues 或電子郵件聯絡我們。
